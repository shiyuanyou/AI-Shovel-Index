# AI Shovel Index — Vision

## 核心问题

普通人很难判断一个新技术概念是否已经进入**投机阶段（卖铲子阶段）**。

当"教程/副业/部署"类商品在二手平台上开始大量涌现，意味着市场情绪已从"好奇"转向"套利"。
这个转变是可以被量化的。

---

## 产品定位

**AI Shovel Index** 是一个每日更新的市场情绪信号源。

- 通过追踪闲鱼上 AI 相关教程类商品的数量变化
- 生成一个 0–100 的热度指数
- 反映当前 AI 技术话题的投机周期位置

信号链：
```
技术发布 → 教程/副业出现 → 卖铲子市场扩大 → 指数上升 → 投机阶段
```

---

## 目标受众

- 关注 AI 技术趋势的观察者
- 社交媒体上的科技内容创作者
- 对市场情绪感兴趣的技术从业者

---

## 核心价值

1. **可传播性**：每日一张图，可直接截图发布社交媒体
2. **连续性**：每日更新，形成时序数据，能看到趋势
3. **话题度**：用数字量化"AI 泡沫"，本身就是话题

---

## 不做什么（边界）

- 不做商业化（无广告、无付费墙、无变现计划）
- 不做实时数据（每日一次已足够）
- 不做 Web 界面（输出物只有图片和文本）
- 不做预测模型（只观测，不预测）
- 不爬取除闲鱼以外的平台（MVP 阶段）

---

## 成功标准（MVP）

- [ ] 能每日自动运行，无需人工干预
- [ ] 生成视觉上可直接发布的 1200×630 图片
- [ ] 生成可直接复制发布的 post.txt 文本
- [ ] 指数计算逻辑清晰、可解释
- [ ] 冷启动（前7天）有合理的降级处理
